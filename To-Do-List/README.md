To-Do List App

A stylish and functional To-Do List web app with priority-based task management and local storage persistence.  
Tasks remain saved even after refreshing or closing the browser.

---

Features
- Add, edit, delete tasks
- Mark tasks as completed
- Assign priority levels (High/Medium/Low) with color-coded indicators
- Tasks automatically sorted by priority
- Data saved in local storage
- Responsive UI with modern design

---

Tech Stack
- HTML5
- CSS(gradient backgrounds, modern layout)
- JavaScript with Local Storage API
