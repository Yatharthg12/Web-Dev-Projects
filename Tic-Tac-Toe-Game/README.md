Tic-Tac-Toe Game

An interactive and responsive Tic-Tac-Toe game built with HTML, CSS, and JavaScript.  
It allows users to play in Player vs Player (PvP) mode or challenge an AI opponent (unbeatable with Minimax algorithm).

---

Features
- Play 2-player mode or vs AI mode
- Intelligent AI opponent that cannot be beaten
- Clean and modern UI design
- Highlights winning combinations
- Mobile-friendly

---

Tech Stack
- HTML
- CSS
- JavaScript
