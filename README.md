3. To-Do List App-
A priority-based task manager that persists data in the browser. Add, edit, delete tasks. Mark tasks as completedAssign and sort by priority (High/Medium/Low). Saves tasks in local storage.

4. Memory Card Game-
A fun memory match game with different difficulty levels. Flip cards to find matching pairs. Tracks moves and time.
Difficulty levels: Easy (4x4), Medium (6x6), Hard (8x8)
Beautiful animated gradient background.
