Smart Calculator

A modern, stylish calculator web application with extended functionality beyond basic arithmetic.  
It supports advanced operations like percentage (%), square root (√), and handles special cases (e.g., division by zero) gracefully.

---

Features
- Clean and responsive UI with professional styling
- Supports addition, subtraction, multiplication, and division
- Extended operations: Percentage (%) and Square Root (√)
- Error handling:
  - Division by zero → 'Error: Division by 0 is undefined'
- Keyboard input support
- Mobile-friendly design

---

Tech Stack
- HTML5
- CSS (modern, responsive styling with gradients and shadows)
- JavaScript
